Quaternion and Rotation

Quaternion

defineï¼š \[ q = a + b\textbf{i} + c\textbf{j} + d\textbf{k}, ~~\textbf{i}^2 =\textbf{j}^2 = \textbf{k}^2 = \textbf{i}\textbf{j}\textbf{k} = -1 \] \(\textbf{i},\textbf{j},\textbf{k}\) are perpendicular to each otherï¼š \[ \textbf{i} \perp \textbf{j} \perp \textbf{k} \] a is perpendicular the 3D space \(\textbf{i}-\textbf{j}-\textbf{k}\)ï¼š \[ a\perp \textbf{i},~~a\perp \textbf{j},~~a\perp \textbf{k} \] letï¼š \[ \vec{v} = b\textbf{i} + c\textbf{j} + d\textbf{k} \] soï¼š \[ q = a + \vec{v} \] defineï¼š \[ unit~~quaternion = \frac{q}{||q||} \]

\[ q q^{-1} = q^{-1} q = 1(unit~~quaternions) \] soï¼š \[ q^{-1} = (a + \vec{v})^{-1} = \frac{(a + \vec{v})^*}{||a+\vec{v}||^2} = \frac{a - \vec{v}}{a^2+||\vec{v}||^2} \]

for each \(q_1 = a_1 + \vec{v_1},q_2 = a_2 + \vec{v_2}\)

additionï¼š \[ q_1 + q_2 = (a_1 + a_2) + (\vec{v_1} + \vec{v_2}) \] multiplicationï¼š \[ q_1 q_2 = (a_1 a_2 - \vec{v_1} \cdot \vec{v_2}) + (a_1 \vec{v_2} + a_2 \vec{v_1} + \vec{v_1} \times \vec{v_2}) \]

Rotation

A rotation of \(\theta\) around the axis \(\vec{u}\)ï¼š \[ \vec{u} = (u_x, u_y, u_z) = u_x\textbf{i} + u_y\textbf{j} + u_z\textbf{k} \] It can be represented by a quaternion \(q\)ï¼š(an extension of Euler's formula.) \[ q = e^{\frac{\theta}{2}(u_x\textbf{i} + u_y\textbf{j} + u_z\textbf{k})} = cos(\frac{\theta}{2}) + (u_x\textbf{i} + u_y\textbf{j} + u_z\textbf{k}) \ sin(\frac{\theta}{2}) \] Given \(p\) in 3D ï¼š(The real part is zero.) \[ \vec{u} = (u_x, u_y, u_z) = 0 + u_x\textbf{i} + u_y\textbf{j} + u_z\textbf{k} \] We can get \(\hat{p}\)ï¼š \[ \hat{p} = q p q^{-1} \] thereï¼š \[ q^{-1} = e^{-\frac{\theta}{2}(u_x\textbf{i} + u_y\textbf{j} + u_z\textbf{k})} = cos(\frac{\theta}{2}) - (u_x\textbf{i} + u_y\textbf{j} + u_z\textbf{k}) \ sin(\frac{\theta}{2}) \]

You can find proof below.

Example

Define the rotation function \(f_r\) around the axis \(\vec{v} = \textbf{i} + \textbf{j} + \textbf{k}\)ï¼Œand the \(\theta = 120^{\circ}\).

Calculate \(q\)ï¼š(normalize \(\vec{v}\)) \[ q = cos(\frac{\theta}{2}) + \frac{\vec{v}}{||\vec{v}||}sin(\frac{\theta}{2}) = \frac{1 + \textbf{i} + \textbf{j} + \textbf{k}}{2} \] So the \(f_r\) isï¼š \[ f_r(a\textbf{i} + b\textbf{j} + c\textbf{k})=q(a\textbf{i}+b\textbf{j}+c\textbf{k})q^{-1} \] thereï¼š \[ q^{-1} = \frac{1 - \textbf{i} - \textbf{j} - \textbf{k}}{2} \] So the actual \(f_r\) isï¼š \[ f_r(a\textbf{i} + b\textbf{j} + c\textbf{k}) = \frac{1 + \textbf{i} + \textbf{j} + \textbf{k}}{2} (a\textbf{i} + b\textbf{j} + c\textbf{k}) \frac{1 - \textbf{i} - \textbf{j} - \textbf{k}}{2} \]

Rotation Proof

Let \(\vec{p}\) around the rotation axis \(\vec{A}\) ratate \(\theta\)ï¼Œthereï¼š \[ ||\vec{A}|| = 1 \] So the projection of \(\vec{p}\) on \(A\) isï¼š \[ p_{A} = cos<\vec{p},\vec{A}> ||\vec{p}|| \vec{A} = \frac{\vec{A} \cdot \vec{p}}{||\vec{p}||}||\vec{p}|| \vec{A} = \vec{A} \vec{p} \vec{A} \] The vertical direction vector isï¼š \[ p_{\perp A} = \vec{p} - \vec{A} \vec{p} \vec{A} \] After rotationï¼Œ\(p_{A}\) don't change.

We can get \(\hat{p}\) by the linear combination of \(p_{\perp A}\) and \(\vec{A} \times \vec{p}\).

\(p_{\perp A}\) rotate around the axis \(A\) with the degree of \(\theta\). \[ p_{\perp A}' = p_{\perp A} cos(\theta) + (\vec{A} \times \vec{p}) sin(\theta) \] So the \(\vec{p}'\) isï¼š \[ \vec{p}' = p_{A} + p_{\perp A}' = \vec{A} \vec{p} \vec{A} +(\vec{p} - \vec{A} \vec{p} \vec{A})cos(\theta) + (\vec{A} \times \vec{p})sin(\theta) \]

\[ \vec{p}' = \vec{p}cos(\theta) + (\vec{A} \times \vec{p})sin(\theta) + (\vec{A} \vec{p} \vec{A})(1 - cos(\theta)) \]

Quaternion Rotation Proof

Let \(q = a + \vec{v}\) is a unit quaternionï¼Œsoï¼š \[ q^{-1} = \frac{a - \vec{v}}{||q||} = a - \vec{v} \] We can prove : \[ \hat{p} = q p q^{-1} = (a + \vec{v}) p (a - \vec{v}) = (a + \vec{v})(0 + \vec{p}) (a - \vec{v})\\ =(-\vec{v} \cdot \vec{p} + a\vec{p} + \vec{v} \times \vec{p})(a - \vec{v}) \\ =a^2\vec{p} + 2a(\vec{v} \times \vec{p}) + (\vec{v} \cdot \vec{p})\vec{v} - \vec{v} \times \vec{p} \times \vec{v} \] Because \(\vec{v} \times \vec{p}\) is perpendicular to \(\vec{v}\) ï¼š \[ \vec{v} \times \vec{p} \cdot \vec{v} = 0 \] andï¼š \[ -a (\vec{p} \times \vec{v}) = a(\vec{v} \times \vec{p}) \]

Theoryï¼šFor each \(\vec{p},\vec{q} \in R^3\)ï¼š \[ \vec{p} \times \vec{q} \times \vec{p} = \vec{p} \times (\vec{q} \times \vec{p}) = \vec{p}^2\vec{q} - (\vec{p} \cdot \vec{q})p \]

By this theoryï¼š \[ - \vec{v} \times \vec{p} \times \vec{v} = (\vec{v} \cdot \vec{p})v - \vec{v}^2\vec{p} \] soï¼š \[ \hat{p} = (a^2 - \vec{v}^2)p + 2a(\vec{v} \times \vec{p}) + 2(\vec{v} \times \vec{p})\vec{v} \] Let \(\vec{v} = t \vec{e},~~||\vec{e}|| = 1\)ï¼Œthenï¼š \[ \hat{p} = (a^2 - t^2)\vec{p} + 2at(\vec{e} \times \vec{p}) + 2t^2(\vec{e} \cdot \vec{p})\vec{e} \] compare withï¼š \[ \vec{p}' = \vec{p}cos(\theta) + (\vec{A} \times \vec{p})sin(\theta) + (\vec{A} \vec{p} \vec{A})(1 - cos(\theta)) \] We can getï¼š \[ \left\{\begin{matrix} a^2 - t^2 = cos(\theta)\\ 2at = sin(\theta)\\ 2t^2 = 1 - cos(\theta) \end{matrix}\right. \] Solve the equation groupï¼š \[ \left\{\begin{matrix} t = \sqrt{\frac{1 - cos(\theta)}{2}} = sin(\frac{\theta}{2})\\ a = cos(\frac{\theta}{2}) \end{matrix}\right. \] soï¼š \[ q = a+\vec{v} = cos(\frac{\theta}{2}) + \frac{\vec{v}}{||\vec{v}||}sin(\frac{\theta}{2}) \] We can use \(q_1 q_2 \cdots q_n\) represent a series of rotation.

Quaternion Rotation Matrix

The rotation \(\hat{p} = q p q^{-1}, q = a + t\vec{e}\) can be represented by a matrix \(\hat{p} = Rp\)ï¼š \[ qpq^-1 = (\begin{bmatrix} a^2 - t^2 & 0 & 0 \\ 0 & a^2 - t^2 & 0 \\ 0 & 0 & a^2 - t^2 \\ \end{bmatrix}+ \begin{bmatrix} 0 & -2at\vec{e}_z & 2at\vec{e}_y \\ 2at\vec{e}_z & 0 & -2at\vec{e}_x \\ -2at\vec{e}_y & 2at\vec{e}_x & 0 \\ \end{bmatrix}+ \begin{bmatrix} 2t^2\vec{e}_x^2 & 2t^2\vec{e}_x\vec{e}_y & 2t^2\vec{e}_x\vec{e}_z \\ 2t^2\vec{e}_x\vec{e}_y & 2t^2\vec{e}_y^2 & 2t^2\vec{e}_y\vec{e}_z \\ 2t^2\vec{e}_x\vec{e}_y & 2t^2\vec{e}_y\vec{e}_z & 2t^2\vec{e}_z^2 \\ \end{bmatrix})p \] let ï¼š \[ q = (w, x, y, z) \] soï¼š \[ \left\{\begin{matrix} w = a\\ x = t\vec{e}_x\\ y = t\vec{e}_y\\ z = t\vec{e}_z\\\end{matrix}\right. \] thenï¼š \[ x^2 + y^2 + z^2 = t^2 \vec{e}^2 = t^2 \] Bring in and simplifyï¼š \[ R = \begin{bmatrix} 1 - 2y^2 - 2z^2 & 2xy - 2wz & 2xz + 2wy \\ 2xy + 2wz & 1 - 2x^2 - 2z^2 & 2yz - 2wx \\ 2xz - 2wy & 2yz + 2wx & 1 - 2x^2 - 2y^2 \end{bmatrix} \]
